# Admin Dashboard

Admin dashboard built with Next.js.

---

## About the Project

### Objective

Build a simple admin dashboard application to manage users and orders.

The project focuses on:

- Clear feature separation
- Realistic admin workflows
- Controlled scope
- Maintainable structure

---

## Project Overview

The dashboard includes the following modules:

- **Authentication**  
  Login page with basic authentication logic, session handled via cookie, protected routes and logout.

- **Users Management**  
  Users list, user creation, deletion and role management (user / admin).

- **Orders Management**  
  Orders list, order detail page and status display.
